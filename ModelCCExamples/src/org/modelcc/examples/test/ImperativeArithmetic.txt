The <b>Imperative Arithmetic language</b> processes assignment and output sentences containing arithmetic expressions in infix notation.<br/>
It features the following capabilities:<br/>
<br/>
- Unary operators: +, and -.<br/>
- Binary operators: +, -, *, and /, being - and / left-associative.<br/>
- Operator priorities: * and / precede + and -.<br/>
- Parenthesized expressions.<br/>
- Integer and floating-point literals.<br/>
- Binary operators: +, and -.<br/>
- Variables: different references to the same variable are links to the same object.<br/>
<br/>
<b>Example input:</b><br/>
<br/>
<pre>
variables
  var a
  var b
  var c
sentences
  a = 1
  b = a+2
  c = (a+b)*2
  output a
  output b
  output c
</pre>
