The Imperative Arithmetic language processes assignment and output sentences containing arithmetic expressions in infix notation.
It features the following capabilities:

- Unary operators: +, and -.
- Binary operators: +, -, *, and /, being - and / left-associative.
- Operator priorities: * and / precede + and -.
- Parenthesized expressions.
- Integer and floating-point literals.
- Binary operators: +, and -.
- Variables: different references to the same variable are links to the same object.
